  • download individual transactions from the archive
  • request archive resubmissions online
  • retrieve on-line reports

To Begin:
  • Wait for the "Continue" button to appear at the bottom of this page. (You may need to scroll down to see the bottom of the page.) This could take up to a minute with a broadband connection or up to 5 minutes over a dial-up connection.
     
  • Important information for Windows XP Service Pack 2 users. You must click on the Information Bar at the top of this page. It reads, "This site might require the following ActiveX control : 'TEAM-UP' from 'Ebix,Inc.' Click Here to install..." Please click on the Information Bar and choose "Install ActiveX Control." When the security warning message appears, click on More Options and choose "Always Install software from Ebix,Inc." Click the Install button.
     
  • If prompted with a dialog box asking if you would like to download and install a control, please click "always trust controls from Ebix, Inc." This will not change the security settings of your machine, but it will allow the download to run properly.
